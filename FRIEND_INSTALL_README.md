# YourMusicSucks Music Bot - Friend Install

## 1) Install

1. Extract this zip.
2. Double-click `install.bat`.
3. Wait for it to finish.

## 2) Configure

1. Open `.env` in a text editor.
2. Fill:
   - `DISCORD_TOKEN`
   - `DISCORD_GUILD_ID`
   - `SPOTIFY_CLIENT_ID`
   - `SPOTIFY_CLIENT_SECRET`

## 3) Run

1. Double-click `run.bat`.
2. Keep that window open while the bot is running.

## Notes

- Commands are slash commands: `/join`, `/play`, `/queue`, `/skip`, `/stop`, `/leave`
- If slash commands do not appear, invite the bot with `applications.commands` scope and restart it.
